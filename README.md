Um sistema Web prático, simples e eficiente para olhar o horário correto e auxiliar ao se organizar com o mesmo, dando uma nova vida aos seus aparelhos antigos.  
A versão atual do sistema e de seus aplicativos é a **Versão 1.0**.

# Descrição

O WatchViewOS (WVOS) é um sistema Web focado em funções baseadas em horas e datas, tentando ser o mais leve possível para executar na maioria dos dispositivos. Com funções simples e de fácil aprendizado, é também voltado para todos os usuários que desejam deixar algum aparelho como relógio principal de algum ambiente.

# Funcionalidades

O WVOS, por padrão, apresenta 6 funcionalidades principais:

- Relógio: Sincronizado pela API [TimeAPI](https://www.timeapi.io/), o sistema sempre terá o seu relógio atualizado de acordo com a região que o usuário escolher. Todas as funções do sistema são baseadas nesse horário, evitando serem ativadas por horários não condizentes com o real.
  
- Menu Home: A forma correta de navegar pelo sistema, você pode apertar o botão secundário do mouse ou apertar e segurar na tela para abrir o Menu Home.

- Alarmes: Programe alarmes para tocar em dias específicos ou todos os dias, com um aviso claro e direto do sistema.
  
- Afazeres: Marque o que você tem que fazer e manipule as suas obrigações de forma prática e rápida.
  
- Eventos: Programe eventos para facilitar a sua vida. O sistema sempre irá alertar no dia que tiver um evento, tanto no relógio principal quanto no Menu Home.

- Configurações: Ajuste tudo do sistema, desde customização até o uso de efeitos no sistema, de forma eficiente, separada e prática.

# Como executar o sistema

Você pode executar o sistema de duas maneiras:

- Acessando [O site do WatchViewOS](https://igorsilva-s.github.io/WatchViewOS)

- Fazendo clone do repositório:
  
  > `git clone https://github.com/IgorSilva-S/WatchViewOS`  
  > Abra o arquivo index.html, dentro da pasta com o nome do sistema

# Iniciando o uso

Ao iniciar o uso do sistema, você será redirecionado para o Setup, um local onde você poderá configurar o sistema da forma que for de sua vontade, além de aprender como utilizar o sistema da melhor forma possível, ou importar dados de outro WatchViewOS para o seu próprio.

# Seus dados

Os desenvolvedores do WatchViewOS não tem acesso algum aos dados do seu sistema e como você o utiliza. Os dados são salvos no próprio navegador do usuário e nada mais. Uma pessoa só terá acesso caso veja os dados do navegador ou receba algum arquivo de importação de dados.
