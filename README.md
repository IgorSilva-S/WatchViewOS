Um sistema Web prático, simples e eficiente para olhar o horário correto e auxilixar ao se organizar com o mesmo, dando uma nova vida aos seus aparelhos antigos.

<h1>Descrição</h1>
O WatchViewOS (WVOS) é um sistema Web focado em funções baseadas em horas e datas, tentando ser o mais leve possível para executar na maioria dos dispositivos. Com funções simples e de fácil aprendizado, é também voltado para todos os usuários que queiram deixar algum aparelho como relógio principal de algum ambiente.

<h1>Funcionalidades</h1>
O WVOS, por padrão, apresenta 4 funcionalidades principais:
- Relógio: Sincronizado pela api <a href="https://www.timeapi.io/">TimeAPI</a>, o sistema sempre terá o seu relógio atualizado de acordo com a região que o usuário escolher. Todas as funções do sistema são baseadas nesse horário, evitando serem ativadas por horários não condizentes com o real.
- Alarmes: Programe alarmes para tocar em dias específicos ou todos os dias
